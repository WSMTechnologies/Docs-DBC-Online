# Admin Panel

Admin panel is the place where you can manage products, categories, coupons, orders, pages, and much more.

# Login

To access the admin panel add /admin after your website address. For example if your website address is "www.dhifaf.online", then your admin panel is located at "www.dhifaf.online/admin".

Now you have to enter your email and password that you've provided. After pressing the Login button, then you'll redirect to the dashboard.

# Dashboard

From the dashboard you'll see an quick overview of DBC Online store.
